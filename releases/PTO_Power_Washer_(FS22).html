About the Mod

A PTO-powered pressure washer designed to clean tools around the farm. In the FS22 version, the washer itself is a decorative piece for realism — to actually use it, you'll also need to buy the separate power washer hose attachment from the shop.

Note

This is a decorative-for-realism build — the power washer needs the separate hose attachment (sold alongside it in-game) to actually be used for cleaning. If you want the fully functional version, the FS25 release uses the game's native high-pressure washer system instead.
