June 2025
WIP Update

Hey everyone, just a quick update on what's been keeping me busy lately. Two main things — a fix coming for the Land Levellers Pack, and the launch of this brand new website.

Land Levellers Pack — Fix in Progress

If you've been using the Land Levellers Pack you may have run into a few issues. I'm aware of them and I'm currently working on an update to get them sorted out.

I don't want to rush it out and introduce new problems, so I'm taking the time to test everything properly before releasing the fix. Once it's ready I'll post an update here with the download link and a full list of what was changed.

💡 In the meantime, if you're hitting a specific issue and need a workaround, feel free to reach out on Discord and I'll try to help.

The New Website

You're looking at it! I recently moved away from Google Sites and built this site from scratch using HTML and CSS, hosted for free on GitHub Pages.

I'd been wanting to do this for a while — Google Sites was fine to get started but it didn't really give me the look and feel I wanted. Building it myself meant I could design it exactly how I had it in my head, and it's been a great way to start learning how web pages actually work.

What's on the site

  • Blog — updates, releases, and general posts like this one
  • Tutorials — step-by-step modding guides aimed at beginners
  • About — a bit about me, the tools I use, and an FAQ

I'll be adding more content regularly as I go. If there's a specific tutorial topic you'd like to see covered, let me know in the Discord.

That's all for now — more soon once the Land Levellers update is ready to go. Thanks for following along!
