What I've Been Working On — July 2026

A rundown of everything currently in progress — some things close to done, some still messy in Blender, and one project that finally shipped.

← Back to Blog
July 2026 · Kelly Modding

Land Levellers Pack — released

The Land Levellers Pack is out now for FS25, on both PC and console, with German and French translations included. This one's been the biggest release in a while and it's good to finally have it off the workbench and into people's hands. If you've grabbed it and run into anything, let me know.

Petrol-Powered Pressure Washer

This is the one eating most of my time at the moment. It started as a high-poly CAD mesh out of Onshape, which looked great but was completely unusable in-game — way too many tris for anything real-time. Rather than just running Decimate and calling it a day (which usually leaves you with ugly, uneven geometry), I've been manually retopologizing it in Blender to get clean, game-appropriate topology while keeping the shape accurate. Slower going, but the result is worth it. The Honda engine model I built a while back is slotting in as the power unit for this once the housing's done.

JM Agri TR Bale Handler

Also modeling a JM Agri TR bale handler at the moment, working off Euro/quick-hitch bracket references pulled from GrabCAD and a few existing FS25 mods to get the mounting geometry right. Still early, but the bracket and frame are taking shape.

Also in the pipeline

  • Bale storage pit — static decorative bale geometry paired with objectStorage for the actual storage logic. Mostly done, just tidying up collisions and re-texturing.
  • Shed mod — in progress alongside everything else.
  • Pajero jeep and a trailer pack — both moving forward in the background.
  • McHale Fusion 2 baler — more of a stretch goal. The plan is a physical bale-count display on the machine itself, driven by a Lua specialization reading balesMadeTotal and mapping it to a UV atlas via setShaderParameter. Technically ambitious, so it's on the back burner until the current batch of mods is out the door.

Other bits

Fixed an Euro Truck Simulator 2 crash that turned out to be a mod conflict in the Scania dealer tab, caused by a corrupted copy of Eugene's Scania NG mod — disabling it sorted it out. Also gave this site a proper pass: everything now runs off one shared stylesheet and script instead of repeating the same code on every page, and there's a new Releases page so all the finished mods are in one place instead of buried in blog posts.

That's the rundown for now — next update should have the pressure washer finished and the bale handler further along.
